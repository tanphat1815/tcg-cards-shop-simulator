# MODULE 5: CƠ SỞ DỮ LIỆU THẺ & THUẬT TOÁN GACHA

## 1. Cấu trúc JSON Dữ liệu (cards.json)
Cần một file static JSON chứa data của Pokémon thật.
Ví dụ cấu trúc 1 thẻ:
```json
{
  "id": "base-001",
  "name": "Bulbasaur",
  "hp": 40,
  "type": "Grass",
  "rarity": "Common",
  "marketPrice": 1.50,
  "imageKey": "card_bulbasaur"
}
