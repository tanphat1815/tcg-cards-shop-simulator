# Naming Convention for Booster Boxes

Thư mục này chứa hình ảnh mặt trước của các hộp bài (Booster Boxes).

## Quy tắc đặt tên (Convention)
Sử dụng **setId** làm tên file chính.

- **Định dạng ưu tiên**: `.webp`
- **Tên file**: `[setId].webp`

### Ví dụ:
- `base1.webp` -> Base Set Booster Box
- `swsh3.webp` -> Darkness Ablaze Booster Box

### Các file đặc biệt:
- `default.webp`: Ảnh hiển thị mặc định nếu không tìm thấy setId tương ứng.
