# Naming Convention for Cards

Thư mục này chứa hình ảnh liên quan đến các lá bài.

## Quy tắc đặt tên (Convention)

### 1. Mặt sau thẻ bài (Card Back)
Đây là hình ảnh quan trọng nhất, hiển thị khi người chơi mở pack.

- **Định dạng**: `.webp` (ưu tiên)
- **Tên file**: `back.webp`
- **Vị trí**: `public/assets/cards/back.webp`

---

### Giới thiệu về Pokemon Cards (Tương lai)
Nếu bạn muốn bổ sung ảnh mặt trước cho từng lá bài cụ thể:
- **Tên file quy ước**: `[cardId].webp` (ví dụ: `base1-1.webp`)
